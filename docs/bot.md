# Telegram Bot

## Назначение

Telegram-бот позволяет быстро добавлять расходы, просматривать отчёты и управлять финансовыми счетами прямо из Telegram — без открытия веб-интерфейса.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие, регистрация пользователя как получателя уведомлений |
| `/help` | Справка по командам |
| `/month` | Отчёт за текущий месяц по категориям |
| `/report` | Псевдоним `/month` |
| `/limits` | Список лимитов по категориям |
| `/setlimit <категория> <сумма>` | Установить лимит. Пример: `/setlimit Еда 1000` |

## Быстрое добавление расхода

Отправь сообщение в формате `Категория Сумма`:

```
Еда 250
Transport 50
```

Бот добавит расход и сообщит об остатке или превышении лимита.

## Inline-меню

После `/start` бот показывает кнопки:
- **Report (month)** — отчёт за месяц
- **add expenses** — выбор категории через клавиатуру с пагинацией
- **Loans & Credit Cards** — финансовое меню

### Финансовое меню

- **Finance Summary** — сводка: доходы (с разбивкой по источникам), долги, минимальные платежи, чистый доход
- **Loans** — список займов с балансом, нажатие → ввод суммы платежа
- **Credit Cards** — список карт с балансом, нажатие → ввод суммы платежа
- **Income** — доходы за текущий месяц с источниками (Denis / Taya / Plaid)

## Finance Summary — формат сообщения

```
💰 Finance Summary (April 2026)

Income:
  👤 Denis: $5,000.00
  👤 Taya: $1,500.00
  🏦 Plaid: $500.00

Min Payments: $750.00
  • Chase Freedom: $25.00
  • Auto Loan: $500.00

Recurring: $250.00

Net Income: $5,000.00

Loans: $25,000.00
Cards: $1,500.00
Total Debt: $26,500.00
```

> Источник **Plaid** показывается с иконкой 🏦, ручные источники — с 👤.
>
> **Net Income** = `income_total − min_payments − recurring_expenses_total`

## Список доходов (Income menu)

Каждая запись показывается с иконкой источника:
- 👤 `Denis — $5,000 — Apr 1`
- 👤 `Taya — $1,500 — Apr 5`
- 🏦 `Plaid — $500 — Apr 3 · Direct Deposit`

## Уведомления

Бот отслеживает пороги расходов по категориям и уведомляет всех зарегистрированных пользователей (peers):
- При первом использовании `/start` пользователь добавляется в список peers
- При добавлении расхода другим пользователем бот отправляет уведомление остальным

## Технические детали

### Доступ

Переменная окружения `ALLOWED_USERS` — список user ID через запятую. Если пуста, бот доступен всем.

```
ALLOWED_USERS=123456789,987654321
```

### Адаптеры

Бот использует два адаптера для работы с API:

**`services/bot_adapter.py`** (async, для legacy операций):
- Расходы, лимиты, отчёты
- Уведомления (peers, threshold alerts)
- Использует `AsyncBudgetApiClient` (aiohttp)

**`services/finance_adapter.py`** (sync, для финансовых операций):
- Займы, карты, платежи, доходы (включая Plaid-доходы)
- При первом вызове делает `POST /api/auth/login` с кредентиалами из env
- Кэширует токен; при истечении (401) автоматически перелогинивается

### Обработка сетевых ошибок

Бот зарегистрирован `error_handler` для `NetworkError` и `TimedOut` от Telegram API. Ошибки логируются на уровне WARNING — бот автоматически повторяет попытки без краша.

### Переменные окружения бота

| Переменная | Назначение | Обязательна |
|------------|------------|-------------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather | Да |
| `ALLOWED_USERS` | Разрешённые user ID через запятую | Нет |
| `API_BASE_URL` | URL FastAPI сервиса | Да |
| `ADMIN_LOGIN` | Логин для авторизации в API | Да (для finance) |
| `ADMIN_PASSWORD` | Пароль для авторизации в API | Да (для finance) |
| `BOT_CURRENCY_SYMBOL` | Символ валюты (по умолч. `$`) | Нет |

### Запуск

```bash
python bot.py
```

На Railway запускается автоматически через `Procfile`:
```
worker: python bot.py
```
