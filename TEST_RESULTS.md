## ⚠️ Тесты, требующие БД

Следующие тесты требуют подключения к PostgreSQL:

- `tests/test_budget_calculations.py` - бюджетные расчеты
- `tests/test_finance_repo.py` - финансовый репозиторий  
- `tests/test_api_endpoints.py` - API endpoints

**Для запуска этих тестов:**
1. Убедитесь, что PostgreSQL запущен
2. Создайте тестовую БД: `createdb budget_pet_test`
3. Установите переменную: `export TEST_DATABASE_URL="postgresql://postgres:postgres@localhost:5432/budget_pet_test"`
4. Запустите: `pytest tests/test_budget_calculations.py`
