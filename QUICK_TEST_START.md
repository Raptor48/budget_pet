# Быстрый старт тестов

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Создание тестовой БД

```bash
createdb budget_pet_test
```

## 3. Запуск тестов

```bash
# Все тесты
pytest

# Или через скрипт
./run_tests.sh
```

## Что проверяется

✅ **Финансовые расчеты** - проценты, графики погашения  
✅ **Бюджетные расчеты** - остатки, переносы между месяцами  
✅ **Финансовый модуль** - CRUD для loans, cards, payments  
✅ **API endpoints** - все HTTP endpoints

## Результаты

- ✅ **Все тесты прошли** = расчеты правильные, все работает
- ❌ **Тесты не прошли** = есть проблемы, нужно исправить

Подробнее: см. `TESTING_GUIDE.md`
